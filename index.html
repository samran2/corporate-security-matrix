Corporate Security Matrix

An open-source, GitHub Pages-ready matrix inspired by the ATT&CK interaction model. The structure is synthesized from leading global official sources including ISO, NIST, WHO, ILO, UNDRR, UNODC, OECD, IFC, ENISA and CISA. EK remains a useful supporting reference, but not the governing framework of the project.

Core logic

Threats and controls in one operating picture

  • Govern cross-functionally with accountable owners and decision rights.
  • Identify critical exposure before it becomes loss.
  • Reduce likelihood through hardening, training and governance.
  • Detect, respond and recover fast enough to preserve continuity.

How to read it

Tabs open ATT&CK-style matrices. Columns stay on the defensive lifecycle.

Each tab opens one domain-specific matrix built as stacked lifecycle columns, similar in interaction logic to the MITRE ATT&CK Enterprise Matrix. It is meant for strategy work, board-level discussion, security program scoping and cross-functional planning across jurisdictions and sectors.

Reference Authorities

Use the strongest official sources as implementation anchors.

  • ISO Risk, resilience, safety, quality and management systems
  • NIST Cybersecurity governance, incident response and control baselines
  • WHO Health emergency management, worker wellbeing and chemical safety
  • ILO Occupational safety, labour protection and workplace conduct
  • UNDRR / UNODC / OECD Resilience, integrity and anti-corruption frameworks
  • IFC Good international industry practice for EHS and site operations
  • ENISA / CISA Threat intelligence, physical security and operational cyber baselines
  • EK Supporting comparator for transferable corporate-security structure

Method

Built from global official frameworks, not a single national model.

Source priority runs from international standards and multilateral guidance to national regulator material. National frameworks such as EK are used as helpful comparators, not as the controlling taxonomy.

Operating Use

Use it as a live management system, not a static poster.

Assign owners to each domain, score lifecycle coverage cell by cell, attach applicable standards and obligations, and refresh the matrix after exercises, incidents and major business change.

ATT&CK-inspired refers to the matrix interaction pattern, not to the MITRE ATT&CK taxonomy. This project is intended as an open-source planning aid for businesses globally, using original synthesis text grounded first in international official sources.
