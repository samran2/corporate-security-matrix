# Code of Conduct

This project is intended to be useful to organizations across sectors, regions and political environments. Keep collaboration professional.

## Expected behavior

- Critique ideas, not people.
- Use evidence when challenging content.
- Prefer clarity over rhetoric.
- Be respectful across cultures, sectors and levels of security maturity.
- Keep contributions practical and grounded in real organizational needs.

## Unacceptable behavior

- Personal attacks or harassment
- Discriminatory language
- Repeated bad-faith argument without sources
- Deliberate misinformation presented as evidence

## Maintainer standard

Maintainers may reject contributions that lower source quality, turn the project into marketing, or make the matrix less globally reusable.
